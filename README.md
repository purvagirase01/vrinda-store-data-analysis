# Vrinda Store Sales Data Analysis (Excel Project)

## 📊 Project Overview
This project analyzes sales data using Microsoft Excel.

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers

## 📌 Key Tasks
- Data Cleaning
- Data Analysis
- Dashboard Creation

## 📈 Insights
- Women customers contribute highest sales
- Age group 30-49 is most active
- Top states: Maharashtra, Karnataka, Uttar Pradesh
