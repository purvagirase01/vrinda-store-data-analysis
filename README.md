# Vrinda Store Sales Data Analysis (Excel Project)

## 📊 Project Overview
This project analyzes sales data using Microsoft Excel.

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers

## 📌 Key Tasks
- Data Cleaning
- Data Analysis
- Dashboard Creation

## 📈 Insights:
-Women customers contributing highest sales
-Adult age group(30-40 yrs) is most active
-Amazon, Flipkart and Myntra channels are max contributing
- Top states: Maharashtra, Karnataka, Uttar Pradesh
